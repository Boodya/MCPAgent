# Skills Directory

Place SKILL.md files here. Each skill should be in its own subdirectory:

```
skills/
├── my-skill/
│   └── SKILL.md
└── another-skill/
    └── SKILL.md
```

SKILL.md format:

```markdown
---
name: skill-name
description: When to use this skill
triggers:
  - keyword1
  - keyword2
---
# Skill Instructions

Detailed instructions for the agent when this skill is activated...
```
